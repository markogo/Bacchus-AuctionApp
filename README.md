# Bacchus ASP.NET Core Web Application

An auction application for everyone!

## Installation

1. Download and install [.NET SDK 5.0](https://dotnet.microsoft.com/download) (If not already installed)
2. Clone this repository
3. Execute the command "dotnet run" from inside the WebApp directory in the command prompt to run the application 
4. Navigate to https://localhost:5001/ in your favourite web browser
